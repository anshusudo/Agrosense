## Visual Elements in PDF

### Color Coding
- **Green (#388e3c)**: Positive metrics, profit areas
- **Blue (#1565c0)**: Key information sections
- **Orange (#FF9800)**: Warnings, loss areas
- **Gray (#f5f5f5)**: Background sections

### Layout Features
- Boxes for emphasis: Risk level, profit, recommendations
- Two-column layouts: For yield and financial data
- Tables: For crop comparison summary
- Bullet points: For loss factors and advice
- Progress indicators: Score representations

### Professional Formatting
- Consistent font sizes (24pt title to 8pt footer)
- Proper spacing and margins
- Line separators for visual organization
- Data aligned in readable columns
- Currency formatting (₹) for Indian market

## Data Accuracy Example

### Scenario: 5-acre rice farm
```
Input Data:
- Farm: 5 acres of Rice on Loamy soil
- Weather: 28°C, 75% humidity, 120mm rainfall

Calculation:
1. Base Yield: 50 q/acre × 5 acres = 250 q
2. Weather Adjustment: 1.0 (good match) = 250 q
3. Loss: (12 + 15 + 8)% = 35% → 87.5 q loss
4. Final Yield: 250 - 87.5 = 162.5 q ≈ 162 q
5. Revenue: 162 × ₹3,100 = ₹502,200
6. Cost: ₹10,000 × 5 = ₹50,000
7. Profit: ₹502,200 - ₹50,000 = ₹452,200
8. Margin: (452,200 / 502,200) × 100 = 90%

Output (in PDF):
- Final Yield: 162 quintals ✓
- Loss: 35% (88 quintals) ✓
- Revenue: ₹502,200 ✓
- Profit: ₹452,200 ✓
- Margin: 90% ✓
```
