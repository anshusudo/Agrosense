Understand the Feature\nThe system analyzes any farm's soil and weather, then:\n- Scores all 8 crops for suitability (0-100%)\n- Selects top 3 crops\n- Calculates yield for each crop\n- Estimates losses (pest, disease, weather)\n- Computes profit/loss for each option\n\n### 2. See It In Action\nBefore implementation, you have two endpoints:\n\n```bash\n# Get recommendation with crop alternatives\nGET /api/recommendation/:farmId\n\n# Generate & email PDF report with crops\nPOST /api/recommendation/:farmId/generate-report\n```\n\n### 3. Review Results\nAPI returns top 3 crops with:\n- Suitability score\n- Expected yield\n- Loss percentage  \n- Net profit in ₹\n- Loss breakdown\n\nPDF report includes:\n- Page 1: Farm & risk details\n- Page 2: Statistics\n- Page 3: **NEW** Alternative crops\n- Page 4: **NEW** Comparison table\n- Page 5: Action items\n\n---\n\n## 🔄 Data Flow Overview\n\n```\nFarmer Requests Recommendation\n        ↓\nSystem gets Farm + Weather\n        ↓\nCurrent Crop Analysis (Fertilizer/Irrigation)\n        ↓\nAlternative Crops Analysis:\n  For each crop:\n    • Score suitability (0-100)\n    • Predict yield\n    • Estimate losses\n    • Calculate profit\n        ↓\nRank by profitability\n        ↓\nReturn API response + Generate PDF\n        ↓\nEmail report or display in UI\n```\n\n---\n\n## 🔑 Key Metrics Explained\n\n### Suitability Score (0-100%)\nHow well the crop matches local conditions:\n- **90-100%**: Excellent fit, high yield expected\n- **75-89%**: Good fit, solid yield\n- **60-74%**: Fair fit, acceptable yield\n- **<60%**: Poor fit, risky choice\n\n### Yield Analysis\n- **Total Yield**: Predicted harvest quantity\n- **Loss %**: Percentage lost to pests/disease/weather\n- **Final Yield**: Actual harvestable quantity\n- **Loss Quintal**: Actual loss in weight\n\n### Financial Metrics\n- **Revenue**: Money from selling harvest\n- **Cost**: Total production expenses\n- **Profit**: Revenue minus cost\n- **Margin**: Profit as % of revenue\n\n---\n\n## 📊 Example: What Farmers See\n\n### In API Response\n```json\n{\n  \"cropRecommendations\": [\n    {\n
